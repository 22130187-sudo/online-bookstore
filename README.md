
## Features

- Browse all books in a clean grid layout
- View book details with description, author, and price
- Add and remove books from a shopping cart
- Adjust quantities in the cart
- Checkout button (frontend simulation)
- Responsive design for all screen sizes
- About and Contact pages
- Navigation bar for easy navigation
